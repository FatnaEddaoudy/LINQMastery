# Sample Resume

This is a placeholder for your resume file. 

**To add your resume:**
1. Save your resume as `resume.pdf` 
2. Place it in the `/public/resume/` folder
3. The download button in the portfolio will automatically work

**To use a different filename:**
1. Update the `resumeFile` property in `/src/data/CVdata.js`
2. Make sure the filename matches exactly
