## üöÄ Quick Start Guide

### Prerequisites
- **Visual Studio 2019+** or **Visual Studio Code** with C# extension
- **.NET Framework 4.7.2+** or **.NET 8.0+**
- **Node.js 16+** (for React/TypeScript projects)
- **Python 3.8+** (for ML projects)
- **Git** for version control

### Getting Started
1. **Clone the repository**
   ```bash
   git clone https://github.com/FatnaEddaoudy/Projects.git
   cd Projects
   ```

2. **Navigate to desired project**
   ```bash
   cd PortfolioProjects/[ProjectName]
   ```

3. **Follow project-specific setup**
   - **C# projects**: Open `.sln` file in Visual Studio
   - **React projects**: `npm install && npm run dev`
   - **Python projects**: `pip install -r requirements.txt && streamlit run app.py`
